How it fits

The pieces reinforce one another.

Author components

Use `.kpa` files so component structure, style, and controller behavior stay close enough to inspect quickly.

Compile with Vite

The Vite plugin translates that component source into runtime-ready modules with explicit contracts.

Route the application

The router owns route metadata, active links, and navigation so the website shell stays predictable.
